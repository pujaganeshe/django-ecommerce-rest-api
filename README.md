# django-ecommerce-rest-api
Full-stack Django eCommerce project featuring REST APIs, shopping cart system, and responsive Bootstrap 

---

## 🚀 Features

- 🛍️ Product Listing
- ➕ Add to Cart
- 🔄 Update Quantity (+ / -)
- ❌ Remove Items from Cart
- 💰 Cart Total Calculation
- 🌐 REST API Integration
- 🎨 Bootstrap UI 

---

## 🛠️ Tech Stack

- Python
- Django
- Django REST Framework
- Bootstrap 5
- SQLite
